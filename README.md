# 📘 Algoritmos e Estruturas de Dados – FCUL

Repositório dedicado aos exercícios, trabalhos práticos e projetos desenvolvidos na disciplina de **Algoritmos e Estruturas de Dados** na Faculdade de Ciências da Universidade de Lisboa.

Este repositório documenta a prática e o aprendizado dos principais conceitos fundamentais da ciência da computação, com foco em modelagem de problemas, eficiência algorítmica e organização de dados.

---

## 🎯 Objetivo

- Consolidar conceitos de algoritmos
- Praticar resolução de problemas computacionais
- Implementar estruturas de dados clássicas
- Desenvolver raciocínio lógico e analítico
- Servir como material de revisão e estudo

---

## 🧠 Conteúdo

O repositório pode incluir implementações e exercícios relacionados a temas como:

- Análise de complexidade (Big-O)
- Recursividade
- Listas, Pilhas e Filas
- Árvores e Grafos
- Algoritmos de ordenação
- Algoritmos de busca
- Estruturas de dados abstratas (ADTs)
- Problemas clássicos de programação
